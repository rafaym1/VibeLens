# homebrew-tap

Homebrew formulae for [CHATS-lab](https://github.com/CHATS-lab) projects.

## Install

```bash
brew install chats-lab/tap/vibelens
```

Or tap first, then install:

```bash
brew tap chats-lab/tap
brew install vibelens
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| vibelens | Agent trajectory analysis and visualization platform |
